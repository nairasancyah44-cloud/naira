# Hijrah Store - Static Web Template

Template statis sederhana untuk toko Hijab Syar'i & Gamis kekinian.
Warna utama: marun (maroon) dan putih.

## Fitur
- Slider produk terlaris (maks 5)
- Tambah ke keranjang & modal keranjang
- Checkout demo (tidak memproses pembayaran)
- Testimoni pelanggan (tersimpan di localStorage)
- Input testimoni
- Form kritik & saran
- Alamat toko: Jl. Dipenggori 74, Bogor

## Cara pakai
1. Ekstrak berkas `hijrah_store.zip`.
2. Upload ke GitHub Pages — cukup taruh isi folder di repository branch `gh-pages` atau `main` (GitHub Pages enabled).
3. Site statis — tidak ada backend. Testimoni & cart disimpan di localStorage browser.

## Struktur folder
```
/css/style.css
/js/script.js
/assets/*.svg
index.html
```
